# Стрингове

Стринговете най-общо представляват текст. Стринговите литерали са текст обграден от:
- единични кавички
  ```python
  string1 = 'This is a single line string!'
  ```
- двойни кавички
  ```python
  string1 = "This is a single line string!"
  ```
- три единични кавички (многоредов стринг)
  ```python
  string1 = '''This is a 
  multi line string!'''
  ```
- три двойни кавички (многоредов стринг)
  ```python
  string1 = """This is a 
  multi line string!"""
  ```
