# Оператори

Python поддържа следните типове оператори:

## Аритметични

- `+`
- `-`
- `/`
- `//`
- `%`
- `*`
- `**`

## За сравнение

- `==`
- `<`
- `>`
- `<=`
- `>=`
- `<>`
- `!=`

## За присвояване

- `=`
- `+=`
- `-=`
- `*=`
- `/=`
- `//=`
- `%=`
- `**=`

## Логически

- `&`
- `|`
- `^`
- `~`
- `<<`
- `>>`

## Побитови

- `and`
- `or`
- `not`

## За Принадлежност (Member)

- `in`
- `not in`

## За Идентичност (Identity)

- `is`
- `is not`
